# Programa de Adopción IA en Tecnología · Edición Claude

Este paquete contiene los archivos base para pedirle a Claude que construya una landing page del programa.

## Archivos incluidos

- `design.md`: dirección visual, narrativa, estructura y requisitos.
- `content.md`: copy completo de la landing.
- `mockdata.ts`: datos estructurados para construir la interfaz.
- `components.md`: arquitectura sugerida de componentes.
- `implementation-prompt.md`: prompt listo para usar con Claude.
- `style-guide.md`: guía visual Proeduca normalizada para este proyecto.

## Nota sobre la fuente

La fuente corporativa debe ser `Proeduca Sans`.

No se incluye ningún archivo de fuente en este paquete.

El proyecto debe estar preparado para usar:

```css
font-family: "Proeduca Sans", Arial, sans-serif;
```

La fuente deberá añadirse manualmente al proyecto.
